Misc¶

Miscellaneous helpful functions.

torch_tools.misc.batch_spatial_dims_power_of_2(batch: Tensor)¶

Check height and width of batch are powers of 2.

Parameters:

batch (Tensor) – A mini-batch of image-like inputs.

Raises:

  • TypeError – If batch is not a Tensor.

  • RuntimeError – If batch does not have four dimensions.

  • RuntimeError – If the batch’s images’ heights are not a power of 2.

  • RuntimeError – If the batch’s images’ heights are not a power of 2.

torch_tools.misc.divides_by_two_check(to_divide: int)¶

Make sure to_divide can be divided by 2.

Parameters:

to_divide (int) – A number to be divided by two.

Raises:

  • TypeError – If to_divide is not an int.

  • ValueError – If to_divide is not greater than zero.

  • ValueError – If to_divide / 2 is irrational.
